
### Algorithm Details

| Alg       | Rotation Type | Activation Clip Ratio | weight quantizer | w/a symetric |
|-----------|---------------|-----------------------|------------------|--------------|
| Quarot    | fixed         | 0.9                   | GPTQ             | yes          |
| FlatQuant | learnable     | learnable             | GPTQ             | yes          |
| Ours      | fixed         | 0.9                   | SignRound        | yes          |

### Eval Details.

lm-eval 0.4.11 for all the data evaluation. 

We report the average of 10 tasks (AVG10) as well as the individual performance on each task. The 10 tasks are: ARC-Challenge (arc_c), ARC-Easy (arc_e), BoolQ (boolq), HellaSwag (hellaswag), Lambada (lambada), MMLU (mmlu), OpenBookQA (obqa), PiQA (piqa), TriviaQA (tfq), and Winogrande (winog).

Following Quarot and FlatQuant, we report `acc_norm` instead of accuracy for tasks where acc_norm is available.

All results below `exclude` KV-cache quantization, as it is not yet supported in our implementation.

### How to Reproduce
~~~ bash
AR_ACT_SCALE=0.9  auto-round Qwen/Qwen3-8B  --scheme INT4 --rotation_type hadamard --format fake --tasks mmlu 
~~~
